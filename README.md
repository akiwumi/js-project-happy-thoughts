## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Usage

- Type your happy thought in the text area
- Watch the character counter as you type (max 140 characters)
- Click "Send Happy Thought" to submit
- Your thought will appear in a card below with a nice animation
- Click the heart icon to like messages

## Validation

The app validates messages:
- Must not be empty
- Must be at least 5 characters long
- Must not exceed 140 characters

Friendly error messages will guide you if validation fails!
